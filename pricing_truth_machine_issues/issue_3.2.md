# ISSUE 3.2 — Query strategy for pricing discovery

**EPIC:** 3 — Real-time internet data retrieval (Tavily)

**Objective:**  
Consistently find pricing-relevant pages.

**Queries:**
- Product pricing context
- Competitors / alternatives pricing
- Prefer URLs with:
  - `/pricing`
  - `/plans`
  - `price`

**DoD:**
- Returns competitor hint URLs when available
- Logs query intent transparently
